# telegram_message_bubble_demo

Telegramdagi messageboxni yasashga harakat qilganimda koʻp qiynaldim. U yerda bir nechta qadamlar qoyilmaqom tarzda
bajarilgan edi.

- Agar xabar qisqa boʻlsa xabar va vaqt bir qatorda 
- Agar xabar uzun boʻlib qatorda joy qolmasa vaqt alohida qatorda
- Agar xabar uzun boʻlib oxirgi qatorda vaqt uchun joy qolsa vaqt oʻsha oxirgi qatorda
- Vaqt doim oʻng taraf qatorning oxirida va asosiy kontentdan biroz pastroqda

Ushbu shartni qanoatlantiradigan holatga olib kelish oson boʻlmadi.
